# Nexus Ultra - Premium Mouse Remapper

![Nexus Ultra](https://img.shields.io/badge/Status-Ultra--Stable-blue?style=for-the-badge)
![License](https://img.shields.io/badge/License-MIT-green?style=for-the-badge)

Nexus Ultra is a high-performance, minimalist mouse remapping engine designed specifically for gaming mice that lack proprietary software. Built with raw Win32 C++ for zero-latency performance and an ultra-light memory footprint.

- **Zero-Latency Remapping**: Direct Raw Input processing for instant button response.
- **Advanced Macro Engine**: Capture and playback complex key sequences with millisecond precision.
- **Ultra Light Service**: Uses less than 8MB of RAM‚Äîlighter than any standard gaming driver.
- **Native Windows Integration**: Built-in "Launch on Startup" and system tray operation.
- **App Launcher**: Map buttons to launch any application or script via a native file picker.
- **Cyberpunk UI**: Modern, glassmorphic interface for configuration.

## üì¶ Installation

1. Download the latest `nexus_ultra_final.exe` from the Releases.
2. Place it in a permanent folder (e.g., `Documents\NexusUltra`).
3. Run the executable.
4. Open the UI (accessible via system tray or local web interface) to configure your buttons.
